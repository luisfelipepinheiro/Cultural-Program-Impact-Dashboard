# Methodology

## Overview

This project evaluates the effectiveness of public cultural programs through a standardized survey completed by participating municipalities.

The methodology was inspired by the principles proposed in the **Brazilian Public Sector Auditing Standards (NBASP 9020)**, adapting key evaluation dimensions into a lightweight assessment framework suitable for Business Intelligence applications.

The objective is to transform qualitative perceptions into measurable indicators that support evidence-based decision making.

---

# Evaluation Process

The evaluation process follows five main steps.

```text
Program Delivery
        │
        ▼
Municipality completes the survey
        │
        ▼
Survey responses are collected
        │
        ▼
Data transformation
        │
        ▼
Power BI Dashboard
        │
        ▼
Impact Indicators
```

---

# Evaluation Dimensions

The survey evaluates seven key dimensions.

| Dimension | Purpose |
|------------|---------|
| Relevance | Measures whether the program addressed local cultural needs. |
| Efficiency | Evaluates whether the program execution met municipal expectations. |
| Effectiveness | Assesses whether the program achieved its immediate objectives. |
| Impact | Measures perceived positive effects on the local community. |
| Reach | Evaluates whether the intended audience was reached. |
| Continuity | Measures the municipality's interest in future editions. |
| Overall Satisfaction | Measures the overall perception of the program. |

---

# Survey Instrument

Each participating municipality answers a standardized questionnaire composed of:

- Seven quantitative questions
- One qualitative question for suggestions and comments

The questionnaire is intentionally short to maximize response rates while providing sufficient information for impact assessment.

---

# Response Scale

All quantitative questions use a five-level evaluation scale.

| Score | Interpretation |
|------:|----------------|
| 1 | Strongly Disagree |
| 2 | Disagree |
| 3 | Neutral |
| 4 | Agree |
| 5 | Strongly Agree |

---

# Qualitative Feedback

The final survey question collects open-ended suggestions.

These responses are intended to support qualitative analysis and identify opportunities for future improvements.

---

```text
Survey Platform
        │
        ▼
Spreadsheet
        │
        ▼
Power Query
        │
        ▼
Star Schema
        │
        ▼
Power BI
```

---

# Expected Outputs

The methodology enables the calculation of indicators such as:

- Overall Satisfaction Score
- Program Effectiveness Index
- Average Score by Evaluation Dimension
- Municipal Participation Rate
- Future Participation Interest

These indicators support evidence-based evaluation of public cultural programs.
