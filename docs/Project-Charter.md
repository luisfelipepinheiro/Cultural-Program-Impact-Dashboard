## Background

Governments often evaluate cultural programs using operational indicators such as budget execution, municipalities served, or participant counts. While useful, these metrics rarely demonstrate whether a program generated meaningful social impact.

This project aims to demonstrate how Business Intelligence can be applied to evaluate the effectiveness of cultural public programs through perception surveys and evidence-based indicators.

---

## Objectives

### Primary Objective

Develop a Business Intelligence dashboard capable of evaluating the effectiveness of public cultural programs.

### Secondary Objectives

- Build an end-to-end BI solution.
- Apply dimensional modeling concepts.
- Demonstrate survey-based impact assessment.
- Create reusable project documentation.
- Publish an open-source portfolio project.
