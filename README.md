# Cultural Program Impact Dashboard

**A Business Intelligence solution for evaluating the effectiveness and social impact of public cultural programs through evidence-based indicators and citizen perception surveys.**

> Measuring impact beyond operational metrics.

---

# About the Project

**Cultural Program Impact Dashboard** is an open-source Business Intelligence project designed to evaluate the effectiveness of public cultural programs using survey-based indicators.

Instead of focusing only on operational metrics, such as the number of municipalities served or participants reached, the project emphasizes impact evaluation by transforming citizen feedback into actionable insights for public managers.

The solution was designed as a lightweight and low-cost BI architecture using accessible technologies such as online survey platforms, Microsoft Excel, Power Query, and Power BI.

The project also demonstrates how dimensional modeling and Business Intelligence can support evidence-based decision making within the public sector.

---

# Why this Project?

Traditional dashboards usually answer questions like:

- How many municipalities were served?
- How many participants attended?
- How much budget was executed?

These indicators describe operational performance but rarely demonstrate whether a public program actually generated meaningful outcomes.

This project was designed to answer a different set of questions:

- Did citizens perceive positive social impact?
- Was access to culture improved?
- Was the initiative well received by the community?
- Should the program be expanded?
- Which aspects should be improved?

By transforming survey responses into structured indicators, the dashboard supports evidence-based public policy evaluation rather than simple operational reporting.

---

# The Problem

Public programs are frequently evaluated using operational indicators such as:

- Number of municipalities served
- Number of participants
- Budget execution
- Number of cultural events

Although these metrics are important, they do not answer a fundamental question:

> **Did the program actually generate positive social impact?**

Without perception-based indicators, decision makers have limited evidence regarding the effectiveness of public cultural policies.

This limitation makes it difficult for public managers to understand whether cultural investments are producing meaningful outcomes for citizens.

---

# The Solution

This project demonstrates how Business Intelligence can support evidence-based public policy evaluation.

Survey responses are collected through online survey platforms, transformed into structured datasets, modeled using a dimensional approach, and visualized through interactive Power BI dashboards.

The result is a practical decision-support framework capable of measuring program effectiveness, citizen satisfaction, and perceived social impact.
